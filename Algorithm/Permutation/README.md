### 순열 (Permutation)

* 1 ~ N 까지 이루어진 수열
* 크기는 항상 N 이어야 하고, 중복된 숫자가 없어야 한다
* 크기가 N 인 순열은 총 N!개가 존재 한다.

#### (사전순으로) 다음 순열을 구하는 알고리즘

순열 A 가 있을때

1. A[i - 1] < A[i] 인 가장 큰 i 를 찾는다. (가장 오른쪽)
2. j >= i 이고, A[j] > A[i - 1] 인 가장 큰 j 를 찾는다. (가장 오른쪽)
3. swap(A[i - 1], A[j])
4. A[i] 부터 reverse 한다.

#### (사전순으로) 이전 순열을 구하는 알고리즘

1. A[i - 1] > A[i] 인 가장 큰 i 를 찾는다. (가장 오른쪽)
2. j >= i 이고, A[j] < A[i - 1] 인 가장 큰 j 를 찾는다. (가장 오른쪽)
3. swap(A[i - 1], A[j])
4. A[i] 부터 reverse 한다.
